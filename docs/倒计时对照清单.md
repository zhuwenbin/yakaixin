# 倒计时对照清单（与小程序一致）

本文档列出 Flutter 中所有倒计时使用位置、用法、样式，并与小程序对应实现对照，确保一致。

---

## 1. 考试剩余时间（答题页）

| 项目 | 小程序 | Flutter | 一致性 |
|------|--------|--------|--------|
| **位置** | `examination/examinationing.vue`、`examination/test.vue` | `exam/views/examinationing_page.dart` | ✅ |
| **文案** | `剩余考试时间：{{ getResidueTime(transformTime(time)) }}` | `剩余考试时间：${_formatTime(state.remainingTime)}` | ✅ |
| **格式** | `transformTime(time)` → `HH:mm:ss`（如 01:23:45） | `_formatTime(seconds)` → `padLeft(2,'0')` 的 HH:mm:ss | ✅ |
| **样式** | `.time`: 24rpx, #2e68ff, 背景 #f5f8ff, 高 64rpx, 居中 | 12.sp, #2E68FF, 背景 #F5F8FF, 32.h 高, 居中 | ✅ |
| **用法** | 每秒由 API/本地计算剩余秒数，getResidueTime 仅处理 NaN 显示 00:00:00 | Provider 每秒递减 remainingTime，_formatTime 格式化为 HH:mm:ss | ✅ |

**小程序参考**：`utils/index.js` 的 `transformTime(time)`；`examinationing.vue` 的 `getResidueTime(time)`。

## 6. 验证码倒计时（登录/忘记密码/修改密码）

## 7. 其他与“倒计时”相关但不改动的部分

- **notice 页开始答题倒计时**：小程序 `examination/notice.vue` 的「开始答题（倒计时：{{ transformTime(time) }}）」为进入考试前的等待，Flutter 若实现同一入口则格式用 HH:mm:ss 即可。
- **intelligentEvaluation 考试倒计时天数**：与学习日历不同业务（计划页「考试倒计时」红色数字），若 Flutter 有对应页再单独对齐。
- **答题/背题中的 countdown 组件**：`commen/countdown.vue`、`questionChallenge/practise.vue` 等为做题限时，与考试剩余时间不同，无需与首页/商品倒计时样式一致。
