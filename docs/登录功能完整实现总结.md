## 🔄 登录后行为

### 1. 返回原页面 ✅

**实现位置**：`login_page.dart` - `_handleLogin()` 方法

**逻辑流程**：
```dart
// ✅ 获取返回路径（优先使用 widget.returnPath）
final returnPath = widget.returnPath ?? 
    (ModalRoute.of(context)?.settings.arguments as String?);

// ✅ 如果有返回路径，跳转到该路径；否则返回上一页
if (returnPath != null && returnPath.isNotEmpty) {
  context.go(returnPath);  // 跳转到原页面
} else {
  Navigator.of(context).pop();  // 返回上一页
}
```

**效果**：
- ✅ 登录成功后，用户返回到触发登录的页面
- ✅ 不会跳转到首页，保持用户的操作上下文

### 场景示例：用户在商品详情页购买

```
1. 用户未登录，浏览商品详情页
   ↓

   ↓

void _showLoginDialog() {
  ConfirmDialog.show(
    context,
    title: '需要登录',
    content: '此操作需要登录账户，请先登录后再试',
    confirmText: '去登录',
    cancelText: '取消',
    onConfirm: () {
      // ✅ 获取当前路径，登录后返回
      final router = GoRouter.of(context);
      final currentPath = router.routerDelegate.currentConfiguration.uri.path;
      context.push(
        AppRoutes.loginCenter,
        extra: {'returnPath': currentPath},
      );
    },
  );
}
```
