# 📱 iOS 内购票据文件位置说明

## 问题背景

之前使用 `Directory.current.path` 在 iOS 运行时返回空路径，导致保存失败：
```
PathNotFoundException: Cannot open file, path = '//lib/features/payment/services/piaoju.txt'
```

## ✅ 解决方案

使用 `path_provider` 包的 `getApplicationDocumentsDirectory()` 获取应用文档目录。

---

## 📂 文件保存位置

### iOS 模拟器

```
/var/mobile/Containers/Data/Application/<UUID>/Documents/piaoju.txt
```

**UUID 说明**:
- 每次安装应用，UUID 会变化
- 需要从日志中复制实际路径
- 示例：`ABC12345-DEFG-6789-HIJK-LMNOPQRSTUV`

### iOS 真机

```
/var/mobile/Containers/Data/Application/<UUID>/Documents/piaoju.txt
```

**访问方式**:
1. Xcode -> Window -> Devices and Simulators
2. 选择设备 -> 选择应用
3. 下载 Container
4. 查看 Documents/piaoju.txt

### Android

```
/data/user/0/com.yakaixin.yakaixin/app_flutter/piaoju.txt
```

**访问方式**:
```bash
adb shell run-as com.yakaixin.yakaixin cat app_flutter/piaoju.txt
```

```
✅ 票据数据已成功写入文件！

   完整路径: /var/mobile/Containers/Data/Application/ABC123.../Documents/piaoju.txt
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**操作步骤**:
1. ✅ 运行应用
2. ✅ 触发购买或清理模式
3. ✅ 搜索日志关键字：`文件路径`

### 方法2: 使用 Xcode Devices

**iOS 模拟器/真机**:
1. Xcode -> Window -> Devices and Simulators
2. 选择设备（模拟器或真机）
3. 选择应用 "YaKaiXin"
4. 点击 ⚙️ (设置) -> Download Container
5. 右键显示包内容 -> AppData -> Documents
6. 找到 `piaoju.txt`

### 方法3: 使用 Dart 代码获取

```dart
import 'package:path_provider/path_provider.dart';

Future<void> printDocumentsPath() async {
  final directory = await getApplicationDocumentsDirectory();
  print('应用文档目录: ${directory.path}');
  print('票据文件路径: ${directory.path}/piaoju.txt');
}
```

## 📝 实现代码

### 修改前（❌ 错误）

```dart
Future<void> _saveReceiptToFile(String receiptData) async {
  // ❌ 在 iOS 运行时 Directory.current.path 返回空字符串
  final currentDir = Directory.current.path;
  final filePath = '$currentDir/lib/features/payment/services/piaoju.txt';
  
  final file = File(filePath);
  await file.writeAsString(receiptData);
}
```

**错误原因**:
- `Directory.current.path` 在 Flutter iOS 应用中返回空路径 `""`
- 导致路径变为 `//lib/features/payment/services/piaoju.txt`
- 系统找不到该路径，抛出 `PathNotFoundException`

### 修改后（✅ 正确）

```dart
import 'package:path_provider/path_provider.dart';

Future<void> _saveReceiptToFile(String receiptData) async {
  // ✅ 使用应用文档目录（iOS/Android 通用）
  final directory = await getApplicationDocumentsDirectory();
  final filePath = '${directory.path}/piaoju.txt';
  
  print('\n💾 ========== 保存票据数据到文件 ==========');
  print('📋 文件路径: $filePath');
  
  final file = File(filePath);
  await file.writeAsString(receiptData);
  
  print('✅ 票据数据已成功写入文件！');
  print('   文件大小: ${receiptData.length} 字符');
  print('   存储位置: 应用文档目录/piaoju.txt');
  print('   完整路径: $filePath');
}
```

**优点**:
- ✅ iOS/Android 通用
- ✅ 系统自动管理目录
- ✅ 应用私有空间，安全
- ✅ 卸载应用时自动清理

## ⚠️ 注意事项

### 1. 路径动态变化

**问题**: iOS 应用沙盒路径包含随机 UUID

**影响**:
- 每次重新安装应用，UUID 会变化
- 不能硬编码路径
- 必须从日志中获取实际路径

**解决方案**:
- 始终使用 `getApplicationDocumentsDirectory()`
- 在日志中打印完整路径
- 从日志复制最新路径

### 2. 文件安全性

**特点**:
- ✅ 存储在应用私有目录
- ✅ 其他应用无法访问
- ✅ 需要越狱或特殊工具才能访问
- ✅ 卸载应用时自动删除

**开发调试**:
- 只能通过 Xcode 或 ADB 访问
- 真机需要下载 Container
- 模拟器可以直接用命令行访问

### 3. 数据持久性

**保留场景**:
- ✅ 应用更新（覆盖安装）
- ✅ 设备重启
- ✅ 应用后台/前台切换

**删除场景**:
- ❌ 应用卸载
- ❌ 清除应用数据（Android）
- ❌ 手动删除文件

### 4. 调试建议

**推荐工具**:
1. **Xcode Devices** - iOS 首选
2. **ADB** - Android 首选
3. **日志查看** - 最简单

**调试步骤**:
1. 运行应用
2. 触发购买或清理
3. 搜索日志：`文件路径`
4. 复制完整路径
5. 使用对应工具查看

## 📊 路径对比表

| 方法 | iOS 路径 | Android 路径 | 优缺点 |
|------|---------|-------------|--------|
| `Directory.current.path` | ❌ 空字符串 `""` | ✅ `/data/user/0/...` | iOS 不可用 |
| `getApplicationDocumentsDirectory()` | ✅ `/var/mobile/Containers/...` | ✅ `/data/user/0/.../app_flutter` | 推荐，跨平台 |
| `getTemporaryDirectory()` | ✅ 临时目录 | ✅ 缓存目录 | 数据可能被清理 |
| `getExternalStorageDirectory()` | ❌ iOS 不支持 | ✅ SD卡目录 | 仅 Android |
