## ✅ 登录后行为

### 1. 返回原页面
- **实现方式**：登录成功后，优先使用 `returnPath` 返回原页面
- **代码位置**：`login_page.dart` - `_handleLogin()` 方法
- **逻辑**：
  ```dart
  // ✅ 如果有返回路径，跳转到该路径；否则返回上一页
  if (returnPath != null && returnPath.isNotEmpty) {
    context.go(returnPath);
  } else {
    Navigator.of(context).pop();
  }
  ```

### 2. 自动刷新数据
- **实现方式**：登录成功后，自动刷新所有主要页面的数据
- **代码位置**：`auth_provider.dart` - `_refreshAllPagesAfterLogin()` 方法
- **刷新的页面**：
  - ✅ 首页（`homeProvider.loadHomeData()`）
  - ✅ 题库页面（`questionBankProvider.loadAllData()`）
  - ✅ 课程页面（`courseNotifierProvider.loadInitialData()`）
- **触发时机**：所有登录方法成功后都会调用
  - `loginWithSms()` - 验证码登录
  - `loginWithPhone()` - 密码登录
  - `loginWithWechat()` - 微信登录

## 📝 实现细节

### 获取当前路径的方法
```dart
final router = GoRouter.of(context);
final currentPath = router.routerDelegate.currentConfiguration.uri.path;
context.push(
  AppRoutes.loginCenter,
  extra: {'returnPath': currentPath},
);
```

### 登录后返回逻辑
```dart
// 在 login_page.dart 中
final returnPath = widget.returnPath ?? 
    (ModalRoute.of(context)?.settings.arguments as String?);

if (returnPath != null && returnPath.isNotEmpty) {
  context.go(returnPath);
} else {
  Navigator.of(context).pop();
}
```

### 登录后刷新逻辑
```dart
// 在 auth_provider.dart 中
void _refreshAllPagesAfterLogin() {
  // 刷新首页
  ref.read(homeProvider.notifier).loadHomeData();
  // 刷新题库
  ref.read(questionBankProvider.notifier).loadAllData();
  // 刷新课程
  ref.read(courseNotifierProvider.notifier).loadInitialData(DateTime.now(), '');
}
```
