### 常见问题排查

#### 问题1: 页面显示空白
**原因**: Mock数据未加载或商品ID不存在
**排查**:
1. 检查日志是否有 `✅ 商品详情数据加载成功`
2. 确认传入的 `goods_id` 在 Mock 数据中存在
3. 检查 `permission_status` 是否为 String 类型

#### 问题2: 跳转路由错误
**原因**: 路由参数传递错误
**排查**:
1. 检查 `home_page.dart` 中的跳转逻辑
2. 确认 `extra` 参数正确传递
3. 查看路由配置 `app_router.dart`

#### 问题3: 图片加载失败
**原因**: Mock数据中的图片URL无效
**解决**:
- 使用真实可访问的图片URL
- 或在Mock数据中使用占位符URL

#### 问题4: 类型转换错误
**原因**: Mock数据类型与Model定义不一致
**解决**:
1. 确保 `type`, `permission_status` 等字段为 String
2. 确保 `prices` 为数组
3. 使用 `SafeTypeConverter` 处理不确定类型
