## 📊 数据字段说明

### 关键字段定义

| 字段 | 类型 | 说明 | 可能值 |
|------|------|------|--------|
| `type` | int | 商品类型 | 2=课程, 8=试卷, 10=模考, 18=章节练习 |
| `data_type` | int | 数据类型 | 2=模考 |
| `details_type` | int | 详情页类型 | 1=经典, 2=真题, 3=科目, 4=模拟 |
| `permission_status` | String | 购买状态 | '1'=已购买, '2'=未购买 |
| `teaching_type` | String | 教学类型 | '1'=直播, '3'=网课 |

### 判断优先级

```
1. 先判断 type (商品类型)
   ├─ type == 2  → 跳转课程详情
   └─ type == 18 → 跳转章节练习 (仅已购买)

2. 再判断 data_type (数据类型)
   └─ data_type == 2 → 模考商品
       └─ 根据 details_type 跳转

3. 最后判断 details_type (详情页类型)
   ├─ details_type == 1 → 经典
   ├─ details_type == 2 → 真题
   ├─ details_type == 3 → 科目
   └─ details_type == 4 → 模拟
```
