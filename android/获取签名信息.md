# 获取 Android 应用签名信息

## 公钥和证书MD5指纹说明

### 什么是公钥？
- **公钥（Public Key）**：Android 应用签名证书的公钥，用于验证应用签名的合法性
- 格式：一串数字或十六进制字符串

### 什么是证书MD5指纹？
- **证书MD5指纹（Certificate MD5 Fingerprint）**：证书的 MD5 哈希值，用于唯一标识证书
- 格式：32位长度的十六进制数字（如：`A1:B2:C3:D4:E5:F6:...`）

---

## 获取方法

### 方法一：从密钥库文件获取（推荐）

```bash
# 进入 android/app 目录
cd android/app

# 查看签名信息（需要输入密钥库密码）
keytool -list -v -keystore release.jks

# 或者只查看 MD5 指纹
keytool -list -v -keystore release.jks | grep -i "MD5"
```

**输出示例：**
```
别名: yakaixin
创建日期: 2025年12月12日
条目类型: PrivateKeyEntry
证书链长度: 0

证书指纹:
     MD5:  A1:B2:C3:D4:E5:F6:...（32位十六进制）
     SHA1: ...
     SHA256: ...
```

**提取公钥：**
```bash
# 导出证书
keytool -exportcert -alias yakaixin -keystore release.jks -file cert.cer

# 查看证书信息（包含公钥）
keytool -printcert -file cert.cer
```

---

### 方法二：从已签名的 APK 获取

```bash
# 使用 jarsigner 查看签名信息
jarsigner -verify -verbose -certs your_app.apk

# 或者使用 apksigner（Android SDK Build Tools）
apksigner verify --print-certs your_app.apk
```

---

### 方法三：使用在线工具

1. 上传 APK 文件到在线工具（如：https://www.apkpure.com/）
2. 查看应用签名信息

---

## 项目信息

- **包名（Package Name）**：`com.yakaixin.yakaixin.android`
- **密钥库文件**：`android/app/release.jks`
- **密钥别名**：`yakaixin`

---

## 注意事项

1. **密钥库密码**：执行 keytool 命令时需要输入密钥库密码
2. **密钥别名**：如果使用不同的别名，需要替换命令中的 `yakaixin`
3. **公钥格式**：备案系统可能需要去除冒号的格式（如：`A1B2C3D4...`）
4. **MD5指纹格式**：通常需要大写字母，冒号分隔（如：`A1:B2:C3:...`）

---

## 快速获取命令

```bash
# 一键获取 MD5 指纹（需要输入密码）
cd android/app && keytool -list -v -keystore release.jks -alias yakaixin | grep -i "MD5" | awk '{print $2}'
```
