## 🔄 验证命令

### 查看正式签名信息
```bash
cd android/app
keytool -list -v -keystore release.jks -alias yakaixin -storepass 135698 -keypass 135698
```

### 查看备用签名信息
```bash
keytool -list -v -keystore ~/.android/debug.keystore -alias androiddebugkey -storepass android -keypass android
```

### 获取MD5指纹
```bash
# 正式签名
keytool -exportcert -alias yakaixin -keystore android/app/release.jks -storepass 135698 -keypass 135698 -file /tmp/cert.cer
openssl x509 -inform DER -in /tmp/cert.cer -noout -fingerprint -md5

# 备用签名
keytool -exportcert -alias androiddebugkey -keystore ~/.android/debug.keystore -storepass android -keypass android -file /tmp/debug_cert.cer
openssl x509 -inform DER -in /tmp/debug_cert.cer -noout -fingerprint -md5
```

## ⚠️ 注意事项

1. **正式签名用于发布**：正式签名（release.jks）用于生产环境发布
2. **备用签名用于开发**：Debug签名仅用于开发和测试
3. **备案使用正式签名**：应用备案时请使用正式签名的信息
4. **MD5格式**：备案系统可能需要去除冒号的格式，请根据系统要求选择
5. **公钥格式**：公钥通常需要完整的Base64字符串，不要包含 `-----BEGIN PUBLIC KEY-----` 和 `-----END PUBLIC KEY-----`
